# SISTEMANG_NEOLIBERAL_PH v1.04
> **Interaktibong Terminal ng Pagkatas ng Buhay**

Isang web-based interactive project na gumagamit ng metapor ng isang dystopian vending machine upang suriin at katawanin ang epekto ng patakarang **Neoliberalismo sa Pilipinas**.

---

## Tungkol sa Proyekto

Sa ilalim ng malayang merkado, ang mga pangunahing karapatan at serbisyong pampubliko tulad ng malinis na tubig, kuryente, transportasyon, at edukasyon ay unti-unting ginagawang komersyal at pribado. Ang proyektong ito ay nagsisilbing salamin kung paanong ang isang ordinaryong mamamayan ay napipilitang "ibenta" ang kanyang oras, labor, at mental na kalusugan upang mabili ang mga bagay na dapat ay batayang karapatan.

### Pangunahing Konsepto (Concept Map Alignment)
* **Audience:** Ang tagagamit ay kinakatawan bilang isang ordinaryong mamamayang Pilipino na nauubusan ng lifespan.
* **Symbolism:** Ang vending machine ay ang sagisag ng pribatisasyon at deregulasyon na naglalayo ng mga karapatan mula sa publiko.
* **Main Idea:** Paglantad sa mga totoong batas at patakaran sa Pilipinas (tulad ng EPIRA Law, Rice Tariffication, at SUC Budget Cuts) na nagpapatakbo sa likod ng sistema.

---

## Tech Stack & Framework
Gawa ang proyekto gamit ang tradisyunal na web framework para sa pixel-perfect retro aesthetics at audio stability:
* **HTML5** – Static structural machine container at audio routing.
* **CSS3** – Retro CRT scanlines visual design, neon electric glowing layouts, at pixel animations.
* **JavaScript (Vanilla)** – Core processing logic, drag-and-drop mechanics, at database mapping.
* **Event System / Animations** – Random matrix character generator at independent screen shaking manipulation.
